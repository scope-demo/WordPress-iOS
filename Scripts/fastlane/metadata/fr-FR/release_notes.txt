- Deux nouveaux blocs sont disponibles ! Bienvenue aux blocs Bouton et Groupe.
- Certains autres blocs ont été améliorés  : Vous pouvez changer les tailles d’images dans les blocs image et utiliser l’option téléverser dans la galerie de blocs. Nous avons aussi pris en charge un problème d’habillage de texte dans les blocs de code court qui coupait des parties de vos contenus.
- Nous avons aussi apporté quelques améliorations à l’expérience générale des blocs. Vous pourrez voir une nouvelle barre d’outils flotter au-dessus du bloc que vous modifiez pour faciliter la navigation dans les blocs, spécialement les plus complexes. De plus, le défilement dans le sélecteur de blocs et dans réglages des blocs est dorénavant pris en charge.
- Enfin, nous avons corriger quelques bugs. En ajoutant une émoji dans le titre d’un article, vous n’aurez plus le code HTML qui se mettait à sa place et l’alignement des blocs de paragraphe est désormais respecté lorsque vous divisez le paragraphe ou lisez le contenu HTML de la publication.
- Nous avons également ajusté les symboles du jour de la semaine en tenant compte de vos réglages régionaux et nous avons ajouté des boutons d’actions rapides aux détails du site pour vous diriger directement vers les parties les plus utilisées de votre site.
